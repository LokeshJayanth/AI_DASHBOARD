# Power BI Template

This directory should contain the `analytics_template.pbix` file.

## How to Create the Template

1. Open Power BI Desktop
2. Create a new report
3. Add the following visualizations:
   - Employee Count by Department (Bar Chart)
   - Average Salary by Department (Bar Chart)
   - Employees Over Time (Line Chart)
   - Salary Distribution (Box Plot)
   - Department Share (Pie/Donut Chart)
4. Connect to a CSV file as the data source
5. Save the file as `analytics_template.pbix` in this directory

## Usage

Users can download this template and:
1. Open it in Power BI Desktop
2. Replace the data source with their cleaned CSV file
3. Refresh the data to see their analytics
